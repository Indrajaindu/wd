圖表繪製

javascript 的圖表繪製函式庫很多，我最常用的兩個是 vis.js 與 c3.js ，其中的 c3.js 是 d3.js 的一個簡易封裝版本。

Vis.js 有 3D 曲線圖，但是互動性不如 d3.js ，但是 d3.js 使用並不容易，封裝為 c3.js 之後就比較容易使用了。

如果您只想選一個使用，我會建議 vis.js 。

  • vis.js : http://visjs.org/
  • c3.js : http://c3js.org/
  • d3.js : http://d3js.org/
